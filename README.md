## Descrição do projeto
O **Olhos Abertos** é uma proposta de aplicativo que tem como objetivo facilitar e agilizar o registro de denúncias de assédio contra mulheres no transporte público da Região Metropolitana do Recife. Ele oferece uma interface rápida, simples e intuitiva, conectando o público-alvo diretamente às delegacias especializadas, a fim de garantir proteção, resposta imediata e mais segurança no dia a dia. Além disso, o aplicativo permite acompanhar o status das denúncias realizadas e conta com a funcionalidade “Botão do Pânico”, que possibilita o contato direto e imediato com a delegacia mais próxima caso a vítima esteja em situação de risco.


## Funcionalidades
- `Funcionalidade 1`: Cadastro de usuário
- `Funcionalidade 2`: Gravação e armazenamento de mídias (fotos, vídeos e áudios)
- `Funcionalidade 3`: Cadastro de denúnica
- `Funcionalidade 4`: Visualização do status das denúncias realizadas
- `Funcionalidade 5`: "Botão do Pânico"
- `Funcionalidade 6`: Aba com informações de canais de apoio a vítima

## Pojeto de banco de dados
Além do protótipo do aplicativo, também desenvolvemos uma simulação do banco de dados associado ao projeto. Nessa etapa, elaboramos a escrita do Minimundo, o modelo conceitual e o modelo lógico utilizando o BrModelo. Por fim, para o modelo físico, utilizamos o MySQL Workbench como ferramenta de desenvolvimento.
